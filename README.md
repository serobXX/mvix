# Mvix CRM Frontend

#### Requirements:
 - node 14
 - yarn
 - ENV variables ( look in the `example.env` )

#### Production build:

```sh
yarn install
yarn run build
```
FE build will be located in the `build` directory.
#### Development:

Configure Prettier in the IDE

```sh
yarn install
yarn run dev
```
After a successful start, the local url will be available in the console.
